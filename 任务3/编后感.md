# 编后感

  在Geek组第三考核阶段，我借助了身边的博客网站和b站视频工具学习了HTML和CSS的基础运用，并且了解了JS的环境，但是对它的运用仍然不熟悉。在短短的三天内，我学会了一些基本的HTML和CSS语法，同时参考一些知名的网页例如京东，豆瓣，淘宝等学习到了一些搭建前端的基本框架，总结反思后我便开始尝试搭建我的第一个静态网页。

- 我构想模仿一个游戏官网的主页面，里面包含了角色信息，攻略，活动公告等内容，基本实现了我的构想，但是还差一些零碎的细节内容比如玩家反馈栏，选择栏等区块。
- 思路：首先我构建了一个主标题定于网页中间，并且利用css工具调整其距离顶部的位置，然后在页面的左上角插入了游戏图标并固定，接着我设计了一块导航栏，这一块设置为右上角固定定位，无论怎么翻动页面都不会移动。然后我在角色图鉴块中插入了不同的图片和文字，活动公告和攻略块我设置了背景颜色和字体大小，模式介绍我利用了表格的形式表达出来，具体形象，设置了左右定位使得页面整洁美观。
- 遇见的难关：
  - 在设计块内容的时候，总是达不到预期的效果，颜色字体排版都经历了很多次尝试，尤其是在插入图片的那一个块，在调整图片之间的间距的时候还要考虑文字的影响，花费了很长的时间，但是好在我跟着模板勉强搭建了一个雏形，后续我觉得需要改进。
  - 在调整块与块的间距的时候，受到学习了的浮动和定位知识的影响，目标块总是不随着我的的指令动，无论怎么调整它都是静止在同一个地方，把代码和效果给了AI后也琢磨了很久还是没有进展，我重新翻看了我的笔记，猜想会不会是块与块的padding关系，尝试了一番有了一定的效果，最终我的攻略和模式介绍得以成型。
  - 最大的问题！：我发现我在放缩页面的时候，我的其他块能正常放缩，而图片和所在的块里面的位置会随着页面跟着变化，导致错位，如今也没有解决，但是正常页面能看，接下来我会查找这方面的资料加以完善；
- 我觉得该项目需要完善和优化，因为背景还是太单调了，没有一点吸引人的特色，因此我会在空余时间完成该项目，作为我的第一个静态网页我还是希望把它做好，让烂尾工程成为高楼大厦。
