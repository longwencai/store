# 页面编后感

在学习完HTML、CSS、JS的知识后，我尝试着复刻Geek招新网站，在历经1天的时间编辑后我感触深悟

1.你构想了一个什么项目，最终完全实现你的构想了吗？

- 我实现了geek招新网站的复刻，95%的内容相似，基本实现了我的构想。

2.你在项目中，是如何用 HTML 给它搭建骨骼、用 CSS 给它增添颜值、用JS给它赋予变化的？

- 我在开始页面设计了一个导航栏，链接到页面的各个部分，接着我插入了图片，设计了与页面比例相似的大小，在接下来的几个栏里面，我采用边框设计、字体颜色大小、文字和图片内容排版，为了让框赋予变化，我设计了让鼠标悬停时图片摆动并放大的效果，使其不再单调。在最后的问题栏处，我利用隐藏的折叠标签，实现了Q&A的互动，并让图片填充了整个块元素。

3.我们总是在关关难过关关过中成长，说说你在项目中遇到的最大难题是啥？你又是如何克服和解决的？

- 在项目中我遇到的最大的问题是在块元素里面元素和文字关系的处理，还有块内块与块的排版，为了使其更美观，我反复尝试了很久，一凡操作之后我向AI寻求了帮助，提供了我想要效果过后，我参照它发给的代码进行了修改。在遇到不会的块标签后我也会向它寻求帮助，最终完成了我的页面。

4.你认为当前的项目还需要优化和补完吗？如果将来你成为一方 dalao，你会想回到最初为这个项目填上最后一块砖吗？

- 我觉得该网页还需要优化，在二维码显示方面我做的还不够好，还有链接的特效也需要改进。另外，JS运用的非常少，我希望能在交互方面做出改进。
