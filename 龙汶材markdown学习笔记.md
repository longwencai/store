# Typora学习笔记

#### Typora是一款支持Markdown语法的文本编辑器，具有极简设计、所见即所得等特点。



## 基础语法

1. 标题：通过“#”加空格实现，如“# 一级标题”，最多支持六级标题，也可使用快捷键“Ctrl+（1~6）”设置。
2. 突出/特殊显示：斜体用“”包裹，加粗用”****“包裹，斜体用"**"包裹，高亮用“==”包裹，删除线用“~~”包裹，下划线快捷键为“Ctrl+U”，上标用“^”，下标用“~”。
3. 列表：无序列表可通过“加号+空格”“减号+空格”或“星号+空格”实现；有序列表通过“数字+点+空格”实现；任务列表语法为“- [ ]”，括号内输入“x”表示勾选完成。
4. 插入内容：表格快捷键为“Ctrl+T”；插入图片语法为“|表头1|表头2|+回车”；超链接语法为“显示文本”；代码块用三个反引号包裹，快捷键为“Ctrl+Shift+K”；公式块用两对美元符号包裹，快捷键为“Ctrl+Shift+M”；分割线可用“*+-”三连符号插入。

### 补充+演示：

1.换行：enter是大换行，shift+enter是小换行；

2.输入乘号，加上\，eg:1\*2\*3

3.无序列表演示：通过“加号+空格”“减号+空格”或“星号+空格”实现 eg:

+ 橘子
+ 苹果（两次enter退出）
  + tab缩进升高一个界面层次
    + 三级分类同理

4.有序列表演示：通过“数字+点+空格”实现 eg:

1. a
2. b
3. c（两次enter退出）

5.任务列表演示：先通过“加号+空格”“减号+空格”或“星号+空格”实现，再加”[空格]空格“ eg:

- [x] (勾选实现任务的实现与否)

6.区块显示：>+回车 eg:

>中国
>
>>武汉
>>
>>>你好（两次enter）

1）行内代码(英文下两次`包裹)

`int a=0;`

2）代码块（三次```）

